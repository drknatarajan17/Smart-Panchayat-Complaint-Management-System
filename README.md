---

# 📌 Project Overview

The **Smart Panchayat Complaint Management System** is an e-Governance application designed to improve communication between citizens and Panchayat authorities.

Citizens can register complaints related to civic issues, while Panchayat officials can assign workers, track complaint status, resolve issues, generate reports, and analyze overall village performance using interactive dashboards.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/drknatarajan17/Smart-Panchayat-Complaint-Management-System.git
```

Go to the project folder

```bash
cd Smart-Panchayat-Complaint-Management-System
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
